# dotfiles

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/).

## Install

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply QuincySx/dotfiles
```

## What's included

- Fish shell config + Dracula theme
- Zsh config
- Starship prompt

## Update

```bash
chezmoi update
```
